# TenderDeve

I build DeFi frontends — trading UIs, liquidity dashboards, and on-chain data visualizations.

React / Next.js + TypeScript + Tailwind, with ethers.js + wagmi + subgraphs underneath. Care about performance, accessibility, and numbers traders trust with real money.

Currently freelancing.

## Now

Building [`perp-trading-ui`](https://github.com/tenderdeve/perp-trading-ui) — perpetual futures DEX interface with TradingView-style candles, live order book depth, and leverage slider with client-side liquidation math. Targeting <150kb gzipped. LP cost-basis tracker is next.
